# AYLA Sapphire (AYLA) 💎

**A transparency-first, deflationary educational token built on Polygon Mainnet.**

AYLA Sapphire is an experimental Web3 project focused on practical blockchain education, on-chain transparency, and verifiable token mechanics.

The project was built from the ground up with a simple philosophy:

* **Don't trust. Verify.**
* **No promises. No guarantees.**
* **Everything is public, auditable, and verifiable directly on-chain.**

## Smart Contract Features
* Fixed supply of 1,000,000,000 AYLA with no `mint()` function
* Strictly deflationary model with publicly verifiable burns
* Transfer fee mechanism with a permanent 5% hard cap
* 24-hour timelock protecting fee and tax collector changes
* Anti-whale transaction and wallet limits
* Separated administrative roles powered by `AccessControl`
* Emergency-only pause mechanism
* ERC-2612 `permit()` support for gasless approvals
* Gas-optimized batch airdrop and token recovery utilities
